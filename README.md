# 議事録タスク抽出ツール

会議の議事録を貼り付けるだけで、やるべきタスクを自動で抽出するPythonツールです。

## 解決する課題

会議後の「メモを読み返してタスクを書き出す作業」に毎日30分かかっていませんか？
このツールに議事録を貼り付けるだけで、担当者・タスク・期限を自動で整理します。

## デモ

```
入力：
・田中さんが来週月曜日までに企画書を作成する
・佐藤さんがクライアントへの連絡を今日中に行う

出力：
- 担当者: 田中 / タスク: 企画書の作成 / 期限: 来週月曜日
- 担当者: 佐藤 / タスク: クライアントへの連絡 / 期限: 当日中
```

## 使用技術

| 技術 | 用途 |
|------|------|
| Python 3.9 | メイン言語 |
| Claude API（claude-sonnet-4-6） | タスク抽出 |
| python-dotenv | APIキー管理 |

## 技術選定の理由

- Claude API：長文の議事録から構造化された情報を安定して抽出できるため
- python-dotenv：APIキーを安全に管理するため

## セットアップ

```bash
git clone https://github.com/あなたのユーザー名/minutes-task-extractor.git
cd minutes-task-extractor
pip3 install anthropic python-dotenv
```

.envファイルを作成してAPIキーを設定：
```
ANTHROPIC_API_KEY=your_api_key_here
```

## 使い方

```bash
python3 tool.py
```

議事録を貼り付けて、空行のままEnterを2回押すとタスクが出力されます。

## 工夫した点

- 担当者が明記されていない場合は「未定」と自動で補完
- 期限の記載がない場合は「期限なし」と明示
- 複数タスクを一括で整理して出力
